## Copy QA Checklist
- Every page has a clear primary CTA when signup is open: `Create account`.
- Fallback waitlist copy appears only when signup is closed.
- No discounts, urgency percentages, or unverifiable proof statements.
- Indie founder path is explicitly framed as primary in homepage and `/for-indie-hackers`.
- Secondary audience pages remain available and coherent, but not dominant in narrative hierarchy.
